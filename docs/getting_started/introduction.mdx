欢迎认识 **MemMachine**，一款面向高级 AI 智能体的开源记忆层。它让 AI 驱动的应用能够在多轮对话之间持续 **学习**、**存储** 和 **回忆** 历史数据与偏好，从而让未来交互更加丰富。MemMachine 的记忆层横跨多个会话、多个智能体以及不同大语言模型，持续构建并更新用户画像，使 AI 聊天机器人升级为真正个性化、具备上下文理解能力的智能助手，提供更精准、更深入的响应。

## 本地或云端部署

准备好正式使用 MemMachine 吗？你可以在本地机器或云环境中运行它。

最简单的方式是使用上方的快速入门选项。如果你希望通过 **pip** 安装或从 GitHub 仓库克隆代码自行部署，可以在左侧导航中的 **安装指南** 找到详细步骤。
