欢迎使用 MemMachine 的评估工具集！我们提供了一套简单的工具，用于衡量 MemMachine 实例的性能、响应质量以及 LoCoMo 评分。

**情景记忆工具集：** 该工具用于评估 MemMachine 在执行核心情景记忆任务时的速度与准确性。具体命令可参考 [Episodic Memory Tool Set](https://github.com/MemMachine/MemMachine/blob/main/evaluation/locomo/episodic_memory/README.md)。


## 快速开始

在运行基准测试之前，请先完成环境准备。

**通用前提条件：**

- **MemMachine 后端：** 所有工具都需要已安装并配置好的 MemMachine 后端。若需要协助，请参阅我们的 [快速入门指南](http://docs.memmachine.ai/getting_started/quickstart)。

- **启动后端：** 完成配置后，使用以下命令启动 MemMachine：

  ```sh
  memmachine-server
  ```

**工具专属前提条件：**

请将你的 `cfg.yml` 复制到 `locomo` 目录（`/memmachine/evaluation/locomo/`）并重命名为 `locomo_config.yaml`。

## 运行基准测试

准备就绪后，按照以下步骤操作。

**A.** 所有命令需在对应工具目录下执行（默认为 `locomo/episodic_memory/`）。

**B.** 请根据文件实际位置，更新数据文件 `locomo10.json` 的路径。默认路径为 `/memmachine/evaluation/locomo/`。

**C.** 完成第 1 步后，可以反复运行第 2-4 步进行测试，最终执行第 5 步清理。

**提示：** 虽然流程简单，但不同工具集的命令可能略有差异。请参考 [Episodic Memory Tool Set](https://github.com/MemMachine/MemMachine/blob/main/evaluation/locomo/episodic_memory/README.md) 获取每一步的具体命令。
