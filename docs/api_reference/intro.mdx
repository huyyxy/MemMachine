## 欢迎

MemMachine 提供多种灵活方式与其强大的记忆层交互。我们将 API 划分为两个主要类别，以满足不同的开发需求：直接集成的 **Python SDK**，以及包含 Web 标准 API 的 **API Playground**。无论你的首选语言或架构是什么，都能顺畅地把 MemMachine 集成到项目中。

## Python SDK（软件开发工具包）

Python SDK 是将 MemMachine 能力直接集成到 Python 应用中的首选方案。它提供一系列预构建的模块和函数，让你在 Python 代码中自然地调用 MemMachine。

- **最适合：** 使用 Python 构建 AI 智能体、数据管道或以 Python 为主的应用。
- **优势：** 在 Python 环境中以最直接、最符合习惯的方式使用 MemMachine 的特性，提供丰富工具并可轻松访问全部功能。

## API Playground

API Playground 是 Web API 的集中入口，涵盖 REST 与 MCP 两类接口。如果你在 Python 之外的环境开发，可以在这里找到连接应用与 MemMachine 的工具。

### REST API（表述性状态转移 API）

REST API 通过简单的 HTTP 请求，以 Web 标准方式与 MemMachine 交互。几乎所有能够发起网络请求的语言或平台都能访问这一通用接口。

- **最适合：** 将 MemMachine 集成到 Web 应用、移动应用或使用任意编程语言（如 JavaScript、Java、Go、C#）的后端服务；同样适合快速测试与脚本编写。
- **优势：** 提供无与伦比的灵活性与互操作性。只要应用可以发起 HTTP 请求，就能与 MemMachine 通信，轻松集成到各种系统。

### MCP API（模型内容协议 API）

MCP API 定义了一个健壮、与语言无关的协议，用于管理所有记忆操作。它为 MemMachine 的记忆层提供高度可扩展、稳定的基础，使不同客户端能够在你的基础设施中安全高效地访问。

- **最适合：** 将 MemMachine 作为独立服务部署、构建微服务架构，或在多应用、多代理（可能使用不同语言编写）共享记忆层的复杂场景中使用。

- **优势：** 为 MemMachine 记忆层提供高度可扩展且稳定的基础，保障不同客户端在你的基础设施中安全高效访问。

请选择最符合你需求的 API，然后在左侧菜单中探索 **API Playground**，了解各类方案如何为你所用。
